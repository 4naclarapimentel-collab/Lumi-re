### RF.005 – Lista de Favoritos

# Lista de Favoritos

**COMO** usuário autenticado da plataforma Lumière,
**QUERO** adicionar filmes à minha lista de favoritos
**PARA** que eu possa encontrar rapidamente os filmes que mais gosto.

### Critérios de Aceitação:

* Deve ser possível adicionar filmes aos favoritos
* Deve ser possível remover filmes da lista de favoritos
* O sistema deve impedir que o mesmo filme seja favoritado mais de uma vez
* O usuário deve conseguir visualizar todos os seus filmes favoritos
