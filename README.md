##  Sobre o Projeto

**Lumière** é um sistema voltado para amantes do cinema, desenvolvido para facilitar a busca, organização e descoberta de filmes de forma prática e moderna.

Inspirado na palavra francesa **“Lumière”**, que significa **luz**, o projeto faz referência à essência do cinema e à história da cinematografia.

A plataforma busca proporcionar uma experiência intuitiva para usuários que desejam explorar títulos, visualizar informações detalhadas, acompanhar avaliações e salvar seus filmes favoritos.

---

- Pesquisa de filmes
-  Catálogo organizado
-  Sistema de avaliações
